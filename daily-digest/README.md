# Daily Digest 格式 v2.0 — 从新闻到 Skill

> 从 2026-06-27 开始，AI 学习日报不再是新闻摘要，而是 "学习 → 行动 → Skill 蒸馏" 的闭环。

---

## 格式原则

1. **1 主题，最多 2 个**
   - 放弃 "6 篇必读" 的广度模式
   - 选择当天最相关、最深刻的 1-2 个点深入

2. **必须回答三个问题**
   - 这个点和我的学习路径哪个阶段相关？
   - 明天我用 AI 时会因此做什么不同？
   - 这个点应该更新/新建哪个 skill？

3. **结构固定为 4 块**
   - 今日主题（1 句话）
   - 学习材料（1-2 篇，附来源）
   - 关键洞见（why it matters）
   - 行动 & Skill 提案（具体、可执行）

---

---

## 与旧版日报的区别

| 维度 | v1.0 新闻式 | v2.0 Skill 式 |
|---|---|---|
| 数量 | 6 篇以上 | 1-2 篇 |
| 深度 | 浅，每篇 1-2 段 | 深，追问 why + so what |
| 价值 | 信息密度 | 行动密度 |
| 输出 | 了解前沿 | 改进 skill / 改变用法 |
| 归档 | 按日期堆放 | 按学习阶段/主题归类 |
